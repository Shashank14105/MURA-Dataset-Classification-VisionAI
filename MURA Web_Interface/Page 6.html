Feature Reuse

DenseNet connects each layer to every other layer in a feed-forward fashion. This alleviates the vanishing-gradient problem and strengthens feature propagation.

Input Adjustments

MURA images vary in size. We resize all inputs to 224×224 pixels and replicate the single grayscale channel to 3 channels to match the ImageNet-pretrained input requirements.

Classification Head

The original 1000-class ImageNet output layer was replaced with a single neuron with Sigmoid activation to output a probability score (0-1) for abnormality detection.
