Deep Learning Pipeline

Methodology

Step 01

Preprocessing

Channel Conversion: Convert single-channel grayscale X-rays to 3-channel RGB (required for pretrained backbones).

Resizing: Standardize input dimensions to 224×224 pixels.

Normalization: Scale pixel values based on ImageNet mean and standard deviation.

Input Transformation

Raw (Various)
224x224x3
Step 02

Data Augmentation

Rotation: Random rotation (±30°) to simulate varied X-ray positioning.

Flipping: Horizontal flips to increase dataset diversity.

Contrast: Random contrast adjustments to handle varied exposure levels.

Augmentation Samples

Step 03

Transfer Learning

Pretrained Weights: Initialize model with ImageNet weights to leverage learned features.

Fine-tuning: Retrain last layers on MURA dataset for specific musculoskeletal features.
