Experimental Setup

Training Strategy & Evaluation

Hyperparameters

Loss Function Binary Cross-Entropy
Optimizer Adam
Learning Rate 1e-4 (0.0001)
Batch Size 16 – 32
Epochs 20 – 50

Strategy: Start with frozen backbone, then unfreeze top layers progressively.

Evaluation Metrics

Accuracy

Overall percentage of correct predictions across all body parts.

Precision & Recall (Sensitivity)

Crucial for minimizing false negatives (missed fractures).

F1 Score

Harmonic mean of precision and recall, useful for class imbalance.

Cohen’s Kappa Score (κ)

The gold standard metric for the MURA dataset.

"Measures inter-rater agreement between the model and radiologists, accounting for the possibility of agreement occurring by chance."

Loss = -(y log(p) + (1-y) log(1-p))

κ = (p_o - p_e) / (1 - p_e)
